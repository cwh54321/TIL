# 파일 관리하기
### path 모듈
path
: 파일과 Directory 경로 작업을 위한 유틸리티로, 경로작업을 할 때 사용

- dirname 함수 : 경로에서 파일 이름을 제외한 경로만 추출

- basename 함수 : 경로에서 파일 이름만 추출

- extname 함수 : 주어진 경로에서 파일의 확장자 추출

- parse 함수 : 경로를 분해해서 각각 객체로 반환 

### FS 모듈
FS 모듈 : 파일의 내용을 읽거나 기록하는 등 파일을 관리할때 사용함

- readdirSync 함수 (동기 처리로 디렉터리 읽기) : 지정한 경로를 읽고 그 경로에 있는 파일 이름을 모두 표시

- readdir 함수 (비동기 처리로 디렉터리 읽기) : 지정한 경로를 읽고 그 경로에 있는 파일 이름을 모두 표시 단 디렉터리를 읽는 동안 시간이 걸리므로 빠르게 처리할 수 있는 다른 일 부터 처리함
